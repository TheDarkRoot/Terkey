# Description

- Utility to add direction keys to Termux.

# Features

- [x] **Supports Android platform.**

# Installation

> ***Open the terminal and type following commands.***
```
$ pkg install git
$ pkg install python
$ git clone https://github.com/TheDarkRoot/Terkey.git
$ cd Terkey
$ chmod +x Terkey.py
$ python Terkey.py
```
